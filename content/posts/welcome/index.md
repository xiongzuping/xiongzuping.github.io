## 网站上线啦！

这是我的新学术主页，使用 [Hugo](https://gohugo.io/) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod) 构建，托管在 GitHub Pages 上。

### 网站功能

- 📄 展示发表的论文
- 📂 介绍研究项目
- ✍️ 分享学术动态
- 🔍 全文搜索
- 🌓 自动切换深色/浅色模式
- 📱 手机电脑都能看

### 如何更新

只需要在 `content/` 目录下用 Markdown 写新内容，push 到 GitHub 后自动发布。
